Steady growth through excellence will be the motto of Scion Software. My goal will be to make everything and everyone I work with better off for it. I intend to grow into the craft of software development and hope to deliver massive value for people along the way.

Open source by default

The nice part about software is that efforts can compound over time to truly make the sum of the work be greater than it's parts. This is why Scion Software will be open source by default. There will obviously be client code bases which will not be open sourced, but the work done by Scion Software for any cross cutting concerns, infrastructure, or generic work will be open and available to those who would either like to use it, or even contribute.

I have a bit of a mission to help the OSS community in .NET blossom. Microsoft has a wonderful framework for efficient development, but we have lagged behind in the open source movement for far too long. With the advent of Microsoft embracing open source in initiatives such as Roslyn, .NET Core and MVC, it is time that we start doing this by default as well.
